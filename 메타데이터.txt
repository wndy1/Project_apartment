# 평균 인구수
 - 표제 : 평균 인구수
 - 제작자 : 자체
 - 유형 : 정수형(integer)
 - 형식 : csv
 - 기간 :  16년 1월 ~ 18년 12월
 - 행정구역 : 인천광역시 모든 구군 모든 읍면동
 - 단위 : 명
 - 출처 : 인천광역시 시청
 - 설명 : 해당 기간(3년)의 행정구역 별 평균 인구 수

# 교육시설 수
 - 표제 : 교육시설 수
 - 제작자 : 자체
 - 유형 : 정수형(integer)
 - 형식 : csv
 - 행정구역 : 인천광역시 모든 구군 모든 읍면동
 - 단위 : 개수
 - 출처 : 인천광역시 교육청
 - 설명 : 각 인천광역시 읍면동에 위치한 초등학교, 중학교, 고등학교의 현황을 파악하고 합한 수
